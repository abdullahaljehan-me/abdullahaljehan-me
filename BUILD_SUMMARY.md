## Getting Started

### Step 1: Add Web3Forms Key (Required for Contact Form)
1. Visit https://web3forms.com and get your free API key
2. Replace `YOUR_WEB3FORMS_ACCESS_KEY` in two places:
   - `main.js` line ~13
   - `contact.html` line ~99

### Step 2: Replace Images
- `assets/profile.jpg` - Your profile photo (400×400px recommended)
- `assets/og-image.jpg` - Social sharing image (1200×630px recommended)

### Step 3: Deploy to GitHub Pages
1. Create repo: `abdullahaljehan-me.github.io`
2. Push all files
3. Enable Pages in repo settings
4. Your site is live at: `https://abdullahaljehan-me.github.io`

### Step 4: Verify & Test
- Visit your live URL
- Test all navigation links
- Try the contact form
- Check mobile responsiveness
- Open browser console for any errors
